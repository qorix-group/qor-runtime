Struct IoxAtomic

#[repr(C)]
pub struct IoxAtomic<T>
where
    T: AtomicInteger,
{ /* private fields */ }
Expand description

iceoryx2 implementation of an atomic that has an internal [RwLockWriterPreference]. It enables atomic operations on platforms that do not support them with the restriction that those operations are no longer lock-free.
