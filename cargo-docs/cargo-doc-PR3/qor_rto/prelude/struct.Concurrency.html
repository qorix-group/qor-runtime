Struct Concurrency

Source 
pub struct Concurrency { /* private fields */ }
Expand description

A Concurrency is an action that executes other actions in multiple branches concurrently. It completes when all concurrent actions have completed. One branch throwing an exception will cause the Concurrency to throw the exception as well.
