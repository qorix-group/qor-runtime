fn check(&self) -> bool

Check if the subscribed event has received a notification since the last check.

fn check_and_reset(&self) -> bool

Check if the subscribed event has received a notification since the last check and reset it.

Reset the event notification state after checking.

Source

fn wait(&self) -> RtoResult<()>

Wait for the event to trigger.

This locks the current thread until the event triggers. The event is reset when this function returns. If the event has already triggered (check would return true), the function returns immediately.

Source

fn wait_timeout(&self, timeout: Duration) -> RtoResult<TimeoutStatus>

Wait for the event to trigger with a timeout.

This locks the current thread until the event triggers or the timeout is reached. The event is reset when this function returns. If the event has already triggered (check would return true), the function returns immediately.
