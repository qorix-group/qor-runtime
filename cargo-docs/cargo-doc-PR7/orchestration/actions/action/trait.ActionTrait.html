Describes action interface that let us build task chain from program. Each action should store it’s actions as Box<dyn ActionTrait> for now

fn execute(&mut self) -> ActionFuture

Will be called on each Program iteration.

Key assumptions: - should avoid allocation except creation of boxed future - each action shall propagate ActionResult down the chain in Future and should immediately stop it’s work once Err(_) is reached, propagating it down.
